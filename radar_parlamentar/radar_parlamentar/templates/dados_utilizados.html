    Extrações da base de dados do Radar

    Os dados listados abaixo são extraídos da base de dados do Radar. Originalmente os dados são oriundos de suas respectivas casas legislativas, mas uma vez importados na base do radar, todos passam a se adequar ao modelo de dados do Radar.

    Ao disponibilizar aqui dados brutos oriundos de diferentes casas legislativas sob formatos unificados, esperamos que outras pessoas possam mais facilmente aplicar análises automatizadas sobre os dados das diferentes casas legislativas.

    Dados completos

    Aqui você pode acessar a última extração realizada com todos os dados da base do Radar. Esses dados estão disponibilizados no formato SQL e são extraídos periodicamente de nossa base de dados em PostgreSQL. Para utilizar esses dados, é preciso importá-los em alguma base de dados com suporte a SQL.

    Download da base completa do Radar aqui (atualizada em {{dumpdate}}).

    Dados específicos

    Abaixo você pode acessar tipos de dados específicos contidos na base do radar, tais como parlamentares, votações, etc. Esses dados estão no formato XML.

    Casas Legislativas

    Estes são dados referentes às casas legislativas analisadas no radar. Nele estão presentes informações como Nome, Esfera e Atualização, entre outros.

    Download

    Legislatura

    Um período de tempo contínuo em que um político atua como parlamentar. É diferente de mandato. Um mandato dura 4 anos. Se o titular sai e o suplente assume, temos aí uma troca de legislatura. Neste arquivo temos informações como Parlamentar, sua Casa Legislativa, Partido e o Início e Fim de sua legislatura.

    Proposição

    Esse arquivo contém os dados relacionados às propostas de leis que serão (ou foram) votadas, como Ementa, Descrição e Situação, entre outras informações.

    Votação e Voto

    Cada proposição possui uma ou mais votações. Em "votação", temos os dados relacionados à proposição, como Data e Resultado da votação. Já em "voto", temos os votos dados por cada parlamentar, assim podemos analisar diretamente cada parlamentar.
